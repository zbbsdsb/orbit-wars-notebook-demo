# Playing Atari with Deep Reinforcement Learning

**Authors**: Volodymyr Mnih, Koray Kavukcuoglu, David Silver, Alex Graves, Ioannis Antonoglou, Daan Wierstra, Martin Riedmiller
**Institution**: DeepMind Technologies
**Year**: 2013
**arXiv**: [1312.5602](https://arxiv.org/abs/1312.5602)

## Core Contribution
First successful combination of deep convolutional neural networks with Q-learning (DQN), enabling an agent to learn directly from raw pixel inputs to play Atari 2600 games at superhuman levels.

## Key Techniques
- **Experience Replay**: Store transitions in a replay buffer and sample random mini-batches to break correlation between consecutive experiences
- **Target Network**: Use a separate slowly-updated network to compute stable Q-value targets, preventing oscillations
- **CNN Architecture**: 3 convolutional layers + 2 fully connected layers to process 84x84x4 input frames
- **Frame Stacking**: Stack 4 consecutive frames as input to capture temporal information
- **Reward Clipping**: Clip rewards to [-1, 1] to stabilize training across different games
- **Epsilon-Greedy Exploration**: Linearly decay exploration rate from 1.0 to 0.1

## Results
- Achieved human-level performance on 29 out of 49 Atari games
- Surpassed all previous RL methods by a large margin
- Demonstrated that a single architecture can learn diverse game strategies

## Relevance to Orbit Wars
- **Experience Replay** is essential for stable RL training in Orbit Wars
- **Target Network** technique should be used in any DQN-based approach
- The general DQN framework is less directly applicable (Orbit Wars uses structured observations, not raw pixels), but the principles of value-based RL are foundational
- Demonstrates that deep RL can master complex game environments from scratch
