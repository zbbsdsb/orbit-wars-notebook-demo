# Soft Actor-Critic: Off-Policy Maximum Entropy Deep Reinforcement Learning with a Stochastic Actor

**Authors**: Tuomas Haarnoja, Aurick Zhou, Pieter Abbeel, Sergey Levine
**Institution**: UC Berkeley
**Year**: 2018
**arXiv**: [1801.01290](https://arxiv.org/abs/1801.01290)

## Core Contribution
An off-policy actor-critic algorithm (SAC) that maximizes both expected reward and entropy. The maximum entropy objective encourages the agent to explore more broadly and makes it robust to model misspecification and estimation errors.

## Key Techniques
- **Maximum Entropy Objective**: Augments the standard RL objective with an entropy term: `J(π) = Σ E[r_t + α H(π(·|s_t))]`
- **Soft Q-Learning**: The critic learns soft Q-values that account for both expected return and policy entropy
- **Automatic Temperature Adjustment**: Learns the entropy coefficient α automatically, eliminating the need for manual tuning
- **Twin Q-Networks**: Uses two Q-functions and takes the minimum to mitigate overestimation bias (similar to TD3)
- **Reparameterization Trick**: Uses a reparameterized policy for low-variance gradient estimation
- **Off-Policy Learning**: Can efficiently reuse data collected by previous policies, improving sample efficiency

## Algorithm
```
# Critic update: minimize soft Bellman residual
y = r + γ * (min(Q_θ1, Q_θ2)(s', a') - α * log π(a'|s'))
Q_loss = E[(Q(s,a) - y)²]

# Actor update: minimize KL divergence
π_loss = E[α * log π(a|s) - min(Q_θ1, Q_θ2)(s, π(s))]

# Temperature update: maintain target entropy
α_loss = E[-α * (log π(a|s) + H_target)]
```

## Results
- State-of-the-art sample efficiency on continuous control benchmarks (MuJoCo)
- More robust to hyperparameter choices than PPO and TD3
- Successfully learned 6-DOF grasping and manipulation tasks
- Outperformed PPO, DDPG, and TD3 on most tasks

## Relevance to Orbit Wars
- **Alternative to PPO** — SAC may perform better if we formulate Orbit Wars with continuous action outputs (e.g., continuous angle and fleet ratio)
- **Sample efficiency** is crucial given limited Kaggle GPU time; SAC's off-policy nature allows reusing past experience
- **Exploration** — maximum entropy helps discover diverse strategies, useful for early training
- **Robustness** — SAC is less sensitive to hyperparameter choices, reducing tuning time
- **Twin Q-networks** technique can be combined with PPO for more stable value estimation
- **Trade-off with PPO**: SAC is better for continuous actions; PPO is better for discrete actions. Orbit Wars uses discrete planet IDs but continuous angles — a hybrid approach may be optimal
