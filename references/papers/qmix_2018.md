# QMIX: Monotonic Value Function Factorisation for Deep Multi-Agent Reinforcement Learning

**Authors**: Tabish Rashid, Mikayel Samvelyan, Christian Schroeder de Witt, Gregory Farquhar, Jakob Foerster, Shimon Whiteson
**Institution**: University of Oxford
**Year**: 2018
**arXiv**: [1803.11485](https://arxiv.org/abs/1803.11485)

## Core Contribution
A value-based multi-agent reinforcement learning algorithm that factorizes the team's joint value function into per-agent value functions using a monotonic constraint. This ensures that the individual agent Q-values are consistent with the global team Q-value while maintaining decentralizability.

## Key Techniques
- **Value Factorization**: Decomposes the global Q-value Qtot into per-agent Q-values Qi such that Qtot = f(Q1, Q2, ..., Qn)
- **Monotonicity Constraint**: The mixing network uses non-negative weights to ensure that each agent's Q-value is monotonically increasing with respect to its individual Q-value. This guarantees IGM (Individual-Global-Max) consistency
- **Mixing Network**: A hypernetwork that takes agent Q-values and global state as input, outputs the team Q-value. Uses absolute value weights followed by softmax to ensure monotonicity
- **Hypernetwork Architecture**: The mixing network's weights are generated by a hypernetwork conditioned on the global state
- **Dueling Architecture**: Each agent uses a dueling network (separate value and advantage streams)
- **Replay Buffer**: Centralized replay buffer storing all agents' transitions

## Algorithm
```
# Mixing network:
w1, w2 = |hypernetwork_1(s)|, |hypernetwork_2(s)|  # non-negative weights
b = hypernetwork_b(s)  # bias from global state
Qtot = Σ(wi * Qi) + b  # monotonic mixing

# Loss:
L(θ) = E[(r + γ * max_a' Qtot(s', a') - Qtot(s, a))²]
```

## Results
- Achieved state-of-the-art on the StarCraft Multi-Agent Challenge (SMAC)
- Solved 5 out of 6 hard SMAC scenarios (previous methods solved 0-2)
- Demonstrated that value factorization is critical for cooperative multi-agent tasks
- Outperformed VDN (Value Decomposition Network) and independent Q-learning

## Relevance to Orbit Wars
- **Cooperative scenarios**: If we train multiple fleets/planets as cooperative agents, QMIX can coordinate them
- **Value decomposition**: The concept of decomposing global value into per-entity values maps to per-planet value estimation
- **Centralized training**: Using global state during training aligns with Orbit Wars' full observability during simulation
- **Less directly applicable** than PPO for competitive settings (QMIX is designed for cooperative tasks), but the value factorization concept is useful
- **SMAC benchmark**: StarCraft multi-agent challenge shares similarities with Orbit Wars (controlling multiple units in real-time)
- **Potential hybrid**: Use QMIX-style value decomposition within a PPO framework for better credit assignment across multiple planets
